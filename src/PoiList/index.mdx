# 主题榜单组件
提供榜单数据，自适应PC,ipad.
暂时不支持 mobile
## Usage
`npm i @ctrip/gs_online_ui`

`import {PoiList} from "gs_online_ui"`


## 效果演示
<PoiList type={2} id={14} source="flight" ENV="UAT"/>



## API
|  参数  | 说明  |  类型 |  默认值 |  必传 | 入参 |
|  ----  | ----  |  ----  | ----  |----  |----  |
| locale  | 语言种类 |  string |  en-US  | N | .. |
| currency  | 货币类型 |  string |  USD  | N | .. |
| ENV  | 环境 |  string |  PROD  | N | .. |
| id  | id |  num |  无  | Y | .. |
|  type | id类型 |  num | 无  | Y | 1=districtId 2=机票cityId  |
|  source | 来源 |  string |  无  | Y | flight |
