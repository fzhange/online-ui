# 主题榜单组件
提供榜单数据，自适应PC,ipad,mobile.
## 引入方式
>import {PoiList} from "gs_online_ui"


## 效果演示
<PoiList  districtId={294} locale="ja-JP"/>



## API

|  参数  | 说明  |  类型 |  默认值 |
|  ----  | ----  |  ----  | ----  |
| locale  | 语言种类 |  string |  en-US  |
| currency  | 货币类型 |  string |  USD  |
| ENV  | 环境 |  string |  PROD  |
| districtId  | 目的地ID |  num |  无  |
